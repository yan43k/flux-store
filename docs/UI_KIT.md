# Flux UI Kit

## Foundations

Flux UI is built from reusable primitives:

- `Button`
- `Card`
- `Badge`
- `Input`
- `SectionHeader`
- `ProductCard`
- `Skeleton`
- `EmptyState`
- `PageShell`

## Visual Tokens

Spacing:

- Page gutter: `1.5rem` mobile, `2rem` tablet, `3rem` desktop.
- Section gap: `5rem`.
- Card padding: `1rem` to `1.5rem`.

Radius:

- Small controls: `0.875rem`.
- Cards: `1.5rem`.
- Hero panels: `2rem`.
- Pills: `999px`.

Shadows:

- Purple glow for primary actions.
- Soft dark elevation for glass panels.
- Inner highlights for premium cards.

## Motion Tokens

- Fast hover: `180ms`.
- Page reveal: `500ms`.
- Stagger delay: `80ms`.
- Easing: spring-like, soft, no abrupt movement.

## Accessibility

- Visible focus states.
- Sufficient contrast on glass backgrounds.
- Buttons and links have clear labels.
- Motion should not be required to understand the UI.

## Layout Patterns

Home:

- Hero with product glow orb.
- Category rail.
- Featured product grid.
- Recommendation bands.
- Brand logos.
- Testimonials, FAQ, newsletter.

Catalog:

- Sticky filters on desktop.
- Drawer filters on mobile.
- Sort and search controls above grid.
- Pagination and loading skeletons.

Product:

- Gallery left, purchase panel right on desktop.
- Specs and reviews below.
- Similar and frequently bought together sections.

Admin:

- Sidebar navigation.
- Stat cards.
- Data tables.
- CRUD forms in dialogs or dedicated pages.
