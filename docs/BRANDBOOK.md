# Flux Brandbook

## Brand Idea

Flux is a premium futuristic electronics store for people who want powerful devices, refined aesthetics, and a fast buying experience. The brand language combines luxury retail clarity with neon cyber-tech energy.

## Logo Concept

The Flux mark is a geometric monogram built from a stylized `F` and a forward energy bolt. It should feel minimal, precise, and high-tech. The symbol works as an app icon, favicon, header logo, and neon badge.

## Color System

Primary palette:

- Neon Purple: `#a855f7`
- Flux Violet: `#7c3aed`
- Dark Purple: `#170826`
- Electric Pink: `#ec4899`
- Deep Space: `#07020f`

Supporting palette:

- Surface Glass: `rgba(255, 255, 255, 0.08)`
- Border Glow: `rgba(168, 85, 247, 0.35)`
- Text Primary: `#faf7ff`
- Text Muted: `#cdbde7`
- Success: `#22c55e`
- Warning: `#f59e0b`
- Danger: `#ef4444`

## Gradients

- Hero Glow: `linear-gradient(135deg, #a855f7 0%, #ec4899 45%, #7c3aed 100%)`
- Card Aura: `radial-gradient(circle at top left, rgba(168,85,247,.35), transparent 35%)`
- Dark Surface: `linear-gradient(180deg, rgba(255,255,255,.10), rgba(255,255,255,.04))`

## Typography

- Headings: Space Grotesk.
- Body: Inter.
- UI controls: Manrope.
- Accents and numeric stats: Sora.

## UI Principles

- Glass surfaces with readable contrast.
- Rounded premium cards.
- Neon borders only for emphasis.
- Motion should feel fast, soft, and intentional.
- Product cards must be commercial and information-rich.
- Dark mode is the flagship theme; light mode is supported as a refined alternative.

## Component Language

- Buttons use gradient fills or glass outlines.
- Cards use soft border glow and hover lift.
- Inputs use glass backgrounds with purple focus rings.
- Badges use compact neon pills.
- Skeletons use animated violet shimmer.
- Empty states include a useful next action.
